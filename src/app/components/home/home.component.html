Basically, a web application was developed that expects a resource file as input from the user and detects a security vulnerability in this resource file and returns. User securely registers, logs in. The logged in user securely initiates the session with the JWT. The user uploads and analyzes the file. As a result, he sees security vulnerabilities. In this way, a full stack application will be developed that works from the bottom up.
